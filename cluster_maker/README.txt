This module provides functions to simulate clusters of data points and export the data to CSV files.

Functions:
    define_dataframe_structure: Define the structure of the DataFrame.
    simulate_data: Simulate data based on the DataFrame structure.
    non_globular_clusters: Simulate non-globular data based on the DataFrame structure.
    export_to_csv: Export the DataFrame to a CSV file.
    export_formatted: Export the DataFrame to a formatted text file.

Demo Files:
    demo1_cm.py: Demonstrate the use of the cluster_maker module with all functions.
    demo2_cm.py: Demonstrate the use of the non_globular_clusters function.
